## Compound Assignment With Augmented Addition
```sh
let a = 35;
let b = 11;
let c = 4;

a += 2;
b += 5;
c += 9;
```
result :
a : 37
b : 16
c : 13

## Compound Assignment With Augmented subtraction
```sh
let a = 1;
let b = 5;
let c = 2;

// Only change code below this line
a -= 1;
b -= 4;
c -= 2;
```
result :
a : 0
b : 1
c : 0
